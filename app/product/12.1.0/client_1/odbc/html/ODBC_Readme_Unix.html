Oracle ODBC Driver for Unix platforms Readme
Release 12.1
December, 2012

 

1 ODBC Driver for Unix platforms

Oracle ODBC Driver for Unix platforms complies with ODBC 3.52 specifications. It is based on features of Oracle 12.1 client. Please refer to Certification Matrix in section 2, to know about the platforms on which 12.1 ODBC driver is supported.
 

1.1   Pre installation task – Install ODBC DM from unixODBC.org

  • Please install ODBC Driver Manager after downloading .tar file from http://www.unixodbc.org/

1.2 Post installation task

  • Configure Oracle ODBC driver on Unix platforms

You can configure Oracle ODBC Driver by running <install-home>/odbc/utl/odbc_update_ini.sh

The utility “odbc_update_ini.sh” takes four command-line arguments

    • arg-1 : Complete path where unixODBC DM has been installed.
    • arg-2 : Complete path of driver install location (optional); if this argument is not passed, the driver path is set to the directory from where the utility is run.
    • arg-3 : Driver name (optional); if this argument is not passed, driver name is set to “Oracle 12c ODBC driver”.
    • arg-4 : Data Source Name (optional); if no value is passed, DSN is set to “OracleODBC-12c”.
  • Update/ verify values of environment variables such as PATH, LD_LIBRARY_PATH, LIBPATH, TNS_ADMIN.

1.3 Uninstalling ODBC Driver

  • Update ~/.odbc.ini file:
    • Remove DSN entry (e.g., "OracleODBC-12c") from [ODBC Data Sources].
    • Remove complete DSN information for corresponding DSN.
  • Update <ODBCDM_HOME>/etc/odbcinst.ini file:
    • Remove driver information for “Oracle 12c ODBC driver”.
  • Remove Oracle ODBC driver for Unix platforms
    • Delete libsqora.so.12.1
  •  Reset environment variables such as PATH, LD_LIBRARY_PATH, LIBPATH, TNS_ADMIN.

1.4 Bugs Fixed

  • Please refer ODBC Release Notes.

 

2 Certification Matrix

We have certified Oracle ODBC driver for release 12.1 against DM 2.3.1 on the following Unix platforms.On 64bit Unix platforms DM 2.3.1 is built with the "-DBUILD_REAL_64_BIT_MODE -DSIZEOF_LONG=8 -fshort-wchar" flags and then certified.

  Platform             32/64 bit       UnixODBC DM ver

  ---------------      ---------       ---------------

  Linux x86-64         32,64 bit           2.3.1

  Solaris SPARC64      32,64 bit           2.3.1

  AIX5L                32,64 bit           2.3.1

  Solaris x64          32,64 bit           2.3.1

  HPUX.IA64            32,64 bit           2.3.1

  ZLinux               32,64 bit           2.3.1

 

Please refer to Installation guide of each platform to know the different flavors of OS and Oracle Client software requirements.
